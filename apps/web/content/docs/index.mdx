**Leezer/UI** 는 [Radix/UI Primitives](https://www.radix-ui.com/) 컴포넌트와 [WAI - ARIA Design Patterns](https://www.w3.org/WAI/ARIA/apg/patterns/) [TailwindCSS](https://tailwindcss.com/) 를 기반으로 제작 되었으며,
이외의 다양한 재사용 가능한 컴포넌트를 제공할 예정입니다.

한글 / 영어 로 문서를 작성해 접근성을 높혀나갈 계획 입니다.

현재는 단순한 UI 요소들이 많지만, 시간을 거쳐서 독창성을 갖는 UI 컴포넌트 라이브러리로 발돋움 하기를 기대합니다.

**컴포넌트 제작기 / 프로젝트 구성 / 라이브러리 분석**에 대한 회고는 [미디엄 블로그](https://medium.com/@2kunhee94/ui-component-library-%EC%A0%9C%EC%9E%91%EA%B8%B0-4b47d4ee5ba) 에서 확인하실 수 있습니다.
